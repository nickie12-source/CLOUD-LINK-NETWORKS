# CLOUD-LINK NETWORKS - ISP Network Design Project

This repository contains the complete documentation and Cisco Packet Tracer simulation for the ISP enterprise network design of **CLOUD-LINK NETWORKS**.

## Project Goal
To design a full ISP network showing connectivity from the **Cloud (Internet backbone)** to **customers**, implementing:

- Redundancy and High Availability
- QoS (Quality of Service)
- VLAN segmentation for 5 departments
- Routing protocols (OSPF and BGP)
- Security rules (ACLs, NAT, firewall policies)
- Network monitoring and management

## Tools Used
- Cisco Packet Tracer
- GitHub
- Draw.io / Visio

## Folder Structure
- `docs/` → documentation files
- `topology/` → packet tracer topology files
- `configs/` → router and switch configurations
- `diagrams/` → logical and physical diagrams
- `reports/` → proposal and final report
- `assets/` → screenshots and images
